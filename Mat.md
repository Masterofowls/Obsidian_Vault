ПОЛНАЯ СТРУКТУРА ОТВЕТОВ НА ВСЕ ВОПРОСЫ

1. Определение погрешности. Источники погрешностей

Определение: Погрешность — количественная мера отклонения приближённого значения от истинного.

Формула: Δx = |x_ист - x_пр|

Формула словами: Абсолютная погрешность равна модулю разности истинного и приближённого значений.

Пояснение к формуле:

· Δ (дельта) — обозначение погрешности
· x_ист — истинное значение (часто неизвестно точно)
· x_пр — полученное приближённое значение
· Модуль | | — делает погрешность всегда положительной

Пример: Истинная длина 15.3 см, измерено 15.0 см → Δ = |15.3 - 15.0| = 0.3 см

Виды источников:

1. Модельные — упрощения в математической модели
2. Методические — погрешности вычислительных методов
3. Вычислительные — ошибки округления при вычислениях
4. Экспериментальные — неточность измерительных приборов

Когда используется: При оценке точности любых измерений и вычислений.

---

2. Абсолютная и относительная погрешность

Определение:

· Абсолютная: отклонение в единицах измерения
· Относительная: доля погрешности от величины

Формулы:

· Δx = |x_ист - x_пр|
· δx = (Δx / |x_ист|) × 100%

Формулы словами:

· Дельта икс равно модуль от (икс истинное минус икс приближённое)
· Дельта малая икс равно (дельта икс делить на модуль икс истинного) умножить на 100 процентов

Пояснение: Абсолютная показывает "насколько ошиблись", относительная — "насколько сильно ошиблись относительно самой величины".

Пример:

· Истинная масса: 100.0 г
· Измерено: 99.5 г
· Δ = 0.5 г, δ = (0.5/100)×100% = 0.5%

Свойства:

· Δ имеет размерность величины
· δ безразмерна (проценты)
· δ позволяет сравнивать точность измерений разных величин

Отличия:

Критерий Абсолютная Относительная
Размерность Есть Нет (проценты)
Сравнение разных величин Нельзя Можно
Оценка качества Неполная Полная

Когда используется:

· Δ — при указании точности приборов
· δ — при сравнении точности разных измерений

3. Значащие цифры

Определение: Все цифры числа от первой ненулевой слева.

Формула: Нет математической формулы, только правила определения.

Правила словами:

1. Все ненулевые цифры — значащие
2. Нули между значащими — значащие
3. Нули в конце после запятой — значащие
4. Нули в начале — незначащие

Пояснение: Значащие цифры показывают, какие цифры в числе несут информацию о его точности.

Примеры:

· 0.0040 → 2 значащие (4 и последний 0)
· 1002 → 4 значащие (1,0,0,2)
· 12.300 → 5 значащих (1,2,3,0,0)

Свойства:

· Количество значащих цифр характеризует точность числа
· Избыточные значащие цифры создают иллюзию точности
· Недостаточные — теряют информацию

Когда используется: При записи результатов измерений и вычислений.

4. Верные значащие цифры в узком и широком смысле

Определение:

· Узкий смысл: цифра верна если Δ ≤ 0.5 единицы её разряда
· Широкий смысл: цифра верна если Δ ≤ 1 единица её разряда

Формулы:

· Узкий: Δ ≤ 0.5 × 10ᵏ
· Широкий: Δ ≤ 1 × 10ᵏ
  где 10ᵏ— единица разряда цифры

Формулы словами:

· Дельта меньше или равно 0.5 умножить на десять в степени к
· Дельта меньше или равно единица умножить на десять в степени к

Пояснение: К — порядок разряда цифры (для десятых k=-1, для сотых k=-2, для единиц k=0 и т.д.)

Пример: Число 23.45, Δ = 0.03

· Для цифры 5 (сотые): единица 0.01 = 10⁻²
· Узкий: 0.03 ≤ 0.5×0.01 = 0.005? НЕТ
· Широкий: 0.03 ≤ 0.01? НЕТ

Отличия:

Критерий Узкий смысл Широкий смысл
Строгость Выше Ниже
Требования Жёстче Мягче
Применение Точные науки Технические расчёты

Когда используется:

· Узкий — в научных исследованиях, метрологии
· Широкий — в инженерных расчётах, технике

5. Правила подсчета цифр

Определение: Правила округления результатов при операциях с приближёнными числами.

Правила:

1. Сложение/вычитание: округлять до разряда наименее точного слагаемого
2. Умножение/деление: округлять до значащих цифр наименее точного числа

Пояснение:

· При сложении складываются абсолютные погрешности → точность определяется наименее точным
· При умножении складываются относительные погрешности → количество значащих цифр определяется наименее точным

Примеры:

1. Сложение: 12.34 + 5.6 = 17.94 → 17.9 (5.6 точнее до десятых)
2. Умножение: 3.14 × 2.0 = 6.28 → 6.3 (2.0 имеет 2 значащие)

Свойства:

· Предотвращают создание иллюзии точности
· Основаны на теории погрешностей
· Эмпирические, но эффективные

Когда используется: При выполнении любых вычислений с приближёнными числами.

6. Метод границ

Определение: Метод оценки погрешности через границы возможных значений.

Формулы:

· x_пр = (a + b)/2
· Δx = (b - a)/2

Формулы словами:

· Приближённое значение равно (а плюс бэ) делить на два
· Погрешность равна (бэ минус а) делить на два

Суть метода: Если известно, что a ≤ x_ист ≤ b, то за наилучшую оценку принимают середину интервала.

Пояснение: a — нижняя граница, b — верхняя граница.

Пример: Длина между 14.7 см и 15.3 см

· x_пр = (14.7 + 15.3)/2 = 15.0 см
· Δx = (15.3 - 14.7)/2 = 0.3 см

Свойства:

· Даёт гарантированную оценку погрешности
· Не требует знания распределения ошибок
· Может давать завышенную оценку

Когда используется: При обработке измерений, когда известны границы возможных значений.

7. Алгебраические и трансцендентные уравнения

Определение:

· Алгебраические: P(x) = 0, где P(x) — многочлен
· Трансцендентные: содержат неалгебраические функции

Формулы:

· Алгебраическое: a_nxⁿ + ... + a₁x + a₀ = 0
· Трансцендентное: F(x) = 0, где F содержит sin, cos, exp, ln и т.д.

Пояснение: Алгебраические описываются конечным набором коэффициентов, трансцендентные — нет.

Примеры:

· Алгебраическое: 2x² - 3x + 1 = 0
· Трансцендентное: sin(x) = x/2

Отличия:

Параметр Алгебраические Трансцендентные
Число корней Конечное Может быть бесконечно
Формулы решения Для n≤4 существуют Нет общих формул
Пример из физики Закон Ома Уравнение колебаний

Свойства алгебраических:

· Теорема Безу: P(a) = 0 ⇔ (x-a) делит P(x)
· Основная теорема алгебры: n корней с учётом кратности

Когда используются:

· Алгебраические — в геометрии, алгебре
· Трансцендентные — в физике, технике, экономике

8. Задача локализации корней

Определение: Нахождение отрезков [a, b], где уравнение f(x) = 0 имеет ровно один корень.

Критерий: f(a) × f(b) < 0

Формула словами: Произведение значений функции на концах отрезка меньше нуля.

Суть метода: Если функция меняет знак на отрезке и непрерывна, то на нём есть корень.

Пояснение: f(a)×f(b) < 0 означает, что f(a) и f(b) имеют разные знаки.

Пример: f(x) = x² - 4

· f(1) = -3 < 0
· f(3) = 5 > 0
  →Корень на [1, 3]

Методы решения:

1. Графический — построение графика
2. Табулирование — таблица значений с шагом
3. Аналитический — исследование свойств функции

Когда используется: Перед применением численных методов уточнения корней.

Свойства:

· Для монотонной функции корень единственный
· Для немонотонной могут быть несколько корней

9. Метод половинного деления (дихотомии)

Определение: Итерационный метод уточнения корня делением отрезка пополам.

Формула: c = (a + b)/2

Формула словами: Середина отрезка равна (а плюс бэ) делить на два.

Суть метода: На каждом шаге отрезок, содержащий корень, делится пополам.

Алгоритм:

1. Начало: [a, b], где f(a)×f(b) < 0
2. c = (a+b)/2
3. Если f(c)=0 → конец
4. Если f(a)×f(c)<0 → b=c, иначе a=c
5. Повторять пока |b-a|>ε

Условия сходимости: f(x) непрерывна на [a, b], f(a)×f(b) < 0

Пример поиска √2 (f(x)=x²-2, ε=0.01):

· [1, 2] → c=1.5
· [1, 1.5] → c=1.25
· [1.25, 1.5] → c=1.375
· ... → 1.414

Когда используется: Для непрерывных функций на известном отрезке.

Свойства:

· Всегда сходится
· Скорость сходимости линейная
· Простой в реализации

10. Метод простых итераций

Определение: Метод решения уравнений вида x = φ(x).

Формула: x_{k+1} = φ(x_k)

Формула словами: Новое приближение равно фи от старого приближения.

Суть метода: Последовательное применение функции φ к начальному приближению.

Условие сходимости: |φ'(x)| < 1 в окрестности корня

Пояснение: φ'(x) — производная, показывает скорость изменения.

Пример: x² - 2 = 0 → x = (x + 2/x)/2
x₀=1→ x₁=1.5 → x₂=1.4167 → ... → 1.4142

Когда используется: Когда уравнение легко преобразуется к виду x = φ(x).

Свойства:

· Может расходиться при неудачном выборе φ(x)
· Скорость сходимости зависит от |φ'(x)|

Отличия от других методов:

· Не требует вычисления производной (как Ньютон)
· Может сходиться быстрее дихотомии

11. Метод хорд

Определение: Метод приближения корня с помощью секущей (хорды).

Формула: x_{новое} = a - f(a)×(b-a)/[f(b)-f(a)]

Формула словами: Новое приближение равно а минус эф от а умножить на (бэ минус а) делить на (эф от бэ минус эф от а).

Суть метода: Аппроксимация функции хордой и нахождение её пересечения с осью X.

Условия: f(a) и f(b) разных знаков, функция непрерывна.

Пример: f(x)=x²-2, [1, 2]

· x₁ = 1 - (-1)×1/(2-(-1)) = 1 + 1/3 = 1.3333
· x₂ = 1.4118 → ... → 1.4142

Когда используется: Когда известен отрезок с корнем, но производную вычислить сложно.

Свойства:

· Сходится всегда при выполнении условий
· Скорость сходимости выше чем у дихотомии
· Не требует вычисления производной

---

12. Метод Ньютона (касательных)

Определение: Метод уточнения корня с помощью касательных.

Формула: x_{n+1} = x_n - f(x_n)/f'(x_n)

Формула словами: Новое приближение равно старое минус эф от старого делить на эф штрих от старого.

Суть метода: Аппроксимация функции касательной в текущей точке.

Условия сходимости:

1. f'(x) ≠ 0 в окрестности корня
2. Начальное приближение достаточно близко к корню

Пример: f(x)=x²-2, f'(x)=2x, x₀=1

· x₁ = 1 - (-1)/2 = 1.5
· x₂ = 1.5 - 0.25/3 = 1.4167
· x₃ = 1.4142

Когда используется: Когда можно вычислить производную и есть хорошее начальное приближение.

Свойства:

· Квадратичная скорость сходимости (быстрее всех)
· Может расходиться при плохом начальном приближении

13. СЛАУ. Матричная форма записи

Определение: Система m линейных уравнений с n неизвестными.

Матричная форма: A·X = B

Формула словами: Матрица коэффициентов умножить на вектор неизвестных равно вектору свободных членов.

Компоненты:

· A = [aᵢⱼ] — матрица коэффициентов m×n
· X = [xⱼ] — вектор неизвестных n×1
· B = [bᵢ] — вектор свободных членов m×1

Пример:
2x+ 3y = 8
4x- y = 2
→
[[2,3], [4, -1]]·[[x], [y]] = [[8], [2]]

Когда используется: В физике, технике, экономике для моделирования линейных систем.

Свойства:

· Решение может быть единственным, бесконечно много или отсутствовать
· Зависит от ранга матриц A и (A|B)

14. Теорема Кронекера-Капелли. Ранг матрицы

Определение ранга: Максимальное число линейно независимых строк (столбцов).

Теорема: СЛАУ совместна ⇔ rang(A) = rang(A|B)

Формулировка словами: Система имеет решение тогда и только тогда, когда ранг основной матрицы равен рангу расширенной матрицы.

Расширенная матрица: (A|B) — матрица A с добавленным столбцом B.

Следствия:

1. rang(A) = rang(A|B) = n → единственное решение
2. rang(A) = rang(A|B) < n → бесконечно решений
3. rang(A) ≠ rang(A|B) → нет решений

Пример: 
x+ y = 3
2x+ 2y = 6
→rang(A)=1, rang(A|B)=1, rang < 2 → бесконечно решений

Когда используется: Для определения существования и количества решений СЛАУ.

15. Метод Гаусса

Определение: Метод решения СЛАУ последовательным исключением неизвестных.

Алгоритм:

1. Прямой ход: приведение к ступенчатому виду
2. Обратный ход: нахождение неизвестных с конца

Суть метода: Элементарные преобразования строк не меняют множество решений.

Элементарные преобразования:

1. Перестановка строк
2. Умножение строки на число ≠ 0
3. Сложение строк

Пример:
2x+ 3y = 8  → (2) - 2×(1): -7y = -14 → y=2
4x- y = 2      из (1): 2x + 6 = 8 → x=1

Когда используется: Для решения СЛАУ небольшой размерности вручную или на компьютере.

Свойства:

· Точный метод (не итерационный)
· Сложность O(n³) для матрицы n×n
· Чувствителен к ошибкам округления

16. Метод Якоби

Определение: Итерационный метод решения СЛАУ.

Формула: xᵢ⁽ᵏ⁺¹⁾ = (bᵢ - Σ aᵢⱼ·xⱼ⁽ᵏ⁾)/aᵢᵢ, j≠i

Формула словами: Новое значение икс и равно (бэ и минус сумма а и жи умножить на икс жи старое) делить на а и и.

Матричная форма: X⁽ᵏ⁺¹⁾ = D⁻¹(B - R·X⁽ᵏ⁾), где A = D + R

Суть метода: Решение каждого уравнения относительно одной переменной.

Пример:
10x+ y = 11 → x = (11 - y)/10
x+ 10y = 11 → y = (11 - x)/10
x₀=0,y₀=0 → x₁=1.1, y₁=1.1 → x₂=0.99, y₂=0.99 → ...

Когда используется: Для больших разреженных систем с диагональным преобладанием.

Свойства:

· Простая реализация
· Медленная сходимость
· Требует диагонального преобладания

---

17. Критерий остановки, условия сходимости метода Якоби

Критерий остановки:

1. max|xᵢ⁽ᵏ⁺¹⁾ - xᵢ⁽ᵏ⁾| < ε (по аргументу)
2. ||A·X⁽ᵏ⁾ - B|| < ε (по невязке)

Формулы словами:

· Максимальное изменение координат меньше эпсилон
· Норма невязки меньше эпсилон

Условия сходимости:

1. Строгое диагональное преобладание: |aᵢᵢ| > Σ|aᵢⱼ|, j≠i
2. Матрица симметричная и положительно определённая

Пояснение: Диагональное преобладание гарантирует, что итерационный процесс сходится.

Пример условия: Для матрицы [[10, 1], [1, 10]]

· |10| > |1| ✓
· |10| > |1| ✓
  →Условие выполняется

Когда используются критерии: На каждой итерации для проверки достижения точности.

18. Постановка задачи аппроксимации

Определение: Приближение заданного набора точек функцией заданного вида.

Математическая формулировка: Найти F(x, a₀, a₁, ..., aₙ) минимизирующую Σ[yᵢ - F(xᵢ)]²

Формула словами: Минимизировать сумму квадратов разностей между заданными значениями и значениями аппроксимирующей функции.

Суть метода: Подбор параметров функции для наилучшего приближения данных.

Пример: Даны точки (1,2), (2,3), (3,5). Найти y = ax + b.

Когда используется:

· Обработка экспериментальных данных
· Сглаживание измерений
· Интерполяция между точками

Методы решения:

1. Метод наименьших квадратов
2. Метод неопределённых коэффициентов
3. Интерполяционные методы

19. Метод неопределённых коэффициентов

Определение: Метод нахождения параметров функции путём решения СЛАУ.

Алгоритм:

1. Выбрать вид функции: F(x) = Σ aᵢ·φᵢ(x)
2. Записать условия: F(xᵢ) = yᵢ для каждой точки
3. Решить полученную СЛАУ относительно aᵢ

Суть метода: Точное прохождение функции через заданные точки.

Пример: Для y = a + bx и точек (0,1), (1,3), (2,7)
a= 1
a+ b = 3 → b = 2
a+ 2b = 7 → проверка: 1+4=5 ≠ 7 → нужно квадратичное приближение

Когда используется: Когда число точек равно числу параметров.

Отличие от МНК:

· Метод неопределённых коэффициентов даёт точное прохождение через точки
· МНК даёт наилучшее приближение в среднем

20. Определённый интеграл. Геометрический смысл

Определение: Предел интегральных сумм при измельчении разбиения.

Формула: ∫[a,b] f(x) dx = lim Σ f(ξᵢ)Δxᵢ при max Δxᵢ → 0

Формула словами: Интеграл от а до бэ от эф от икс дэ икс равно предел суммы эф от кси и умножить на дельта икс и.

Геометрический смысл: Площадь криволинейной трапеции под графиком f(x) на [a, b].

Пояснение: Δxᵢ — длина i-го отрезка разбиения, ξᵢ — точка на этом отрезке.

Пример: ∫[0,2] x² dx — площадь под параболой y=x² от 0 до 2.

Когда используется:

· Вычисление площадей, объёмов
· Решение физических задач (работа, путь)
· Теория вероятностей

Свойства:

· ∫[a,b] f(x) dx = -∫[b,a] f(x) dx
· ∫[a,b] [f(x)+g(x)] dx = ∫[a,b] f(x) dx + ∫[a,b] g(x) dx

21. Формула прямоугольников

Определение: Приближённое вычисление интеграла суммой площадей прямоугольников.

Формулы:

1. Левые: S ≈ h·Σ f(xᵢ), i=0..n-1
2. Правые: S ≈ h·Σ f(xᵢ), i=1..n
3. Средние: S ≈ h·Σ f((xᵢ+xᵢ₊₁)/2)

Формулы словами: Шаг умножить на сумму значений функции в точках.

Пояснение: h = (b-a)/n — шаг разбиения, n — число отрезков.

Пример: ∫[0,1] x² dx, n=4, h=0.25

· Левые: 0.25×[0² + 0.25² + 0.5² + 0.75²] = 0.21875
· Точное значение: 1/3 ≈ 0.3333

Когда используется: Для быстрой приближённой оценки интеграла.

Точность: Погрешность ~ O(h) для левых/правых, O(h²) для средних.

22. Формула трапеций

Определение: Приближённое вычисление интеграла суммой площадей трапеций.

Формула: S ≈ h·[f(x₀)/2 + f(x₁) + ... + f(xₙ₋₁) + f(xₙ)/2]

Формула словами: Шаг умножить на (полусумма крайних значений плюс сумма внутренних значений).

Вывод формулы: Площадь i-й трапеции = h·(f(xᵢ)+f(xᵢ₊₁))/2, суммируем все.

Пример: ∫[0,1] x² dx, n=4, h=0.25
S= 0.25×[0/2 + 0.0625 + 0.25 + 0.5625 + 1/2] = 0.34375

Когда используется: Для более точного чем прямоугольники вычисления интегралов.

Точность: Погрешность ~ O(h²) — в два раза точнее прямоугольников.

Преимущества перед прямоугольниками:

· Учитывает наклон функции на отрезке
· Обычно даёт лучшую точность при том же n

9. Метод половинного деления (дихотомии)

Определение: Численный метод нахождения корня уравнения f(x)=0 путём последовательного деления отрезка пополам.

Формула: c = (a + b)/2

Формула словами: Новая точка c равна полусумме концов отрезка a и b.

Суть метода: На каждом шаге отрезок [a,b], содержащий корень, делится пополам, и выбирается та половина, где функция меняет знак.

Пояснение к формуле:

· a, b — концы текущего отрезка
· c — середина отрезка
· Знак f(a)·f(c) определяет, в какой половине искать корень

Алгоритм:

1. Проверить: f(a)·f(b) < 0
2. Вычислить c = (a+b)/2
3. Если f(c)=0 → корень найден
4. Если f(a)·f(c) < 0 → b = c, иначе a = c
5. Повторять пока |b-a| > ε или |f(c)| > ε

Пример (поиск √2 через x²-2=0, ε=0.001):

```
Итерация 1: [1, 2] → c=1.5, f(1.5)=0.25>0 → новый отрезок [1, 1.5]
Итерация 2: [1, 1.5] → c=1.25, f(1.25)=-0.4375<0 → [1.25, 1.5]
Итерация 3: [1.25, 1.5] → c=1.375, f(1.375)=-0.1094<0 → [1.375, 1.5]
...
После 10 итераций: c≈1.4142 (√2≈1.41421356)
```

Свойства:

1. Всегда сходится при выполнении начальных условий
2. Скорость сходимости: линейная (погрешность уменьшается в 2 раза за итерацию)
3. Количество итераций: n ≥ log₂((b-a)/ε)
4. Устойчивость: не зависит от вида функции (только от непрерывности)

Когда используется:

· Когда известен отрезок, содержащий корень
· Для непрерывных функций
· Когда требуется гарантированная сходимость
· В начальных этапах решения для грубой локализации корня

Преимущества:

· Простота реализации
· Гарантированная сходимость
· Не требует вычисления производной

Недостатки:

· Медленная сходимость
· Требует знания отрезка с корнем
· Не эффективен для функций с несколькими корнями близко друг к другу

10. Метод простых итераций

Определение: Итерационный метод решения уравнений вида x = φ(x).

Формула: x_{k+1} = φ(x_k)

Формула словами: Каждое следующее приближение вычисляется как φ от предыдущего.

Суть метода: Уравнение f(x)=0 преобразуется к виду x = φ(x), затем строится последовательность x₀, φ(x₀), φ(φ(x₀)), ...

Пояснение к формуле:

· x_k — текущее приближение
· x_{k+1} — следующее приближение
· φ(x) — итерационная функция

Условие сходимости: |φ'(x)| < 1 в окрестности корня x*

Пример (решение x²-2=0):

1. Выбор φ(x):
   · Вариант 1: x = 2/x (плохо: |φ'(√2)| = 2/2 = 1)
   · Вариант 2: x = (x + 2/x)/2 (хорошо: |φ'(√2)| = 0)
2. Итерации с x₀=1:
   ```
   x₁ = (1 + 2/1)/2 = 1.5
   x₂ = (1.5 + 2/1.5)/2 ≈ 1.4167
   x₃ = (1.4167 + 2/1.4167)/2 ≈ 1.4142
   ```

Критерий остановки: |x_{k+1} - x_k| < ε или |f(x_k)| < ε

Свойства:

1. Скорость сходимости: линейная при 0<|φ'(x*)|<1
2. Может расходиться при неудачном выборе φ(x)
3. Чувствительность к начальному приближению

Когда используется:

· Когда уравнение легко преобразуется к виду x = φ(x)
· Для уточнения корня после грубой локализации
· В комбинации с другими методами

Виды итерационных функций:

1. Явная итерация: x = φ(x) непосредственно
2. Релаксация: x = x + ω·f(x), где ω — параметр релаксации
3. Метод Ньютона как частный случай: φ(x) = x - f(x)/f'(x)

Практические рекомендации:

· Проверять условие сходимости |φ'(x)| < 1
· Начинать с приближения, близкого к корню
· Использовать разные формы φ(x) для тестирования сходимости

11. Метод хорд

Определение: Метод приближённого решения уравнений с помощью линейной интерполяции (секущей).

Формула (для неподвижного левого конца):
x_{новое}= a - f(a)·(b-a)/[f(b)-f(a)]

Формула словами: Новое приближение = левый конец минус f(левый конец) умножить на длину отрезка, делить на разность значений функции на концах.

Суть метода: Функция аппроксимируется хордой (прямой через точки (a,f(a)) и (b,f(b))), корень хорды берётся как новое приближение.

Пояснение к формуле:

· a, b — концы текущего отрезка
· f(a), f(b) — значения функции на концах
· (b-a)/[f(b)-f(a)] — угловой коэффициент обратной функции

Алгоритм:

1. Выбрать отрезок [a,b] с f(a)·f(b)<0
2. Вычислить x₁ по формуле
3. Если f(a)·f(x₁)<0 → b=x₁, иначе a=x₁
4. Повторять пока |f(x)|>ε

Пример (решение x²-2=0 на [1,2]):

```
Итерация 1: x₁ = 1 - (-1)·1/(2-(-1)) = 1 + 1/3 ≈ 1.3333
Итерация 2: f(1.3333)≈-0.2222, новый отрезок [1.3333,2]
           x₂ = 1.3333 - (-0.2222)·0.6667/(2-(-0.2222)) ≈ 1.4118
Итерация 3: x₃ ≈ 1.4142
```

Свойства:

1. Скорость сходимости: сверхлинейная (~1.618 - золотое сечение)
2. Всегда сходится для непрерывных функций на отрезке с корнем
3. Не требует вычисления производной

Когда используется:

· Когда вычисление производной сложно или невозможно
· Для монотонных функций на известном отрезке
· В комбинации с методом Ньютона (метод секущих)

Модификации метода:

1. Метод ложного положения: всегда держит корень в интервале
2. Метод секущих: использует две последние точки (не фиксирует концы)

Отличия от метода Ньютона:

· Не требует производной
· Медленнее сходится
· Более устойчив к плохому начальному приближению

12. Метод Ньютона (касательных)

Определение: Итерационный метод решения уравнений с использованием производной.

Формула: x_{n+1} = x_n - f(x_n)/f'(x_n)

Формула словами: Новое приближение = старое минус значение функции, делённое на значение производной.

Суть метода: Функция аппроксимируется касательной в текущей точке, корень касательной берётся как новое приближение.

Пояснение к формуле:

· x_n — текущее приближение
· f(x_n) — значение функции
· f'(x_n) — значение производной
· f(x_n)/f'(x_n) — поправка к приближению

Геометрическая интерпретация:
Касательная в точке(x_n, f(x_n)): y = f(x_n) + f'(x_n)(x - x_n)
Её пересечение с осью OX:0 = f(x_n) + f'(x_n)(x - x_n) → x = x_n - f(x_n)/f'(x_n)

Пример (решение x²-2=0):

```
f(x)=x²-2, f'(x)=2x
x₀=1: x₁ = 1 - (-1)/2 = 1.5
x₁=1.5: x₂ = 1.5 - 0.25/3 ≈ 1.4167
x₂=1.4167: x₃ = 1.4167 - 0.0069/2.8334 ≈ 1.4142
```

Условия сходимости:

1. f'(x) ≠ 0 в окрестности корня
2. f''(x) непрерывна
3. Начальное приближение достаточно близко к корню: |x₀ - x| < 2|f'(x)|/|f''(x*)|

Критерий остановки: |x_{n+1} - x_n| < ε или |f(x_n)| < ε

Свойства:

1. Скорость сходимости: квадратичная (погрешность ~ квадрату предыдущей)
2. Чувствительность к начальному приближению
3. Требует вычисления производной

Когда используется:

· Когда можно легко вычислить производную
· Для быстрого уточнения корня
· В оптимизации (поиск экстремумов)

Модификации:

1. Модифицированный метод Ньютона: f'(x) вычисляется один раз
2. Метод Ньютона для систем уравнений: требует матрицы Якоби
3. Метод Ньютона-Рафсона: для алгебраических уравнений

Преимущества:

· Быстрая сходимость
· Простота реализации для простых функций

Недостатки:

· Может расходиться при плохом начальном приближении
· Требует вычисления производной
· Не работает при f'(x)=0

13. Система линейных алгебраических уравнений (СЛАУ). Матричная форма записи

Определение: Совокупность m линейных уравнений с n неизвестными.

Общий вид:

```
a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ = b₁
a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ = b₂
...
aₘ₁x₁ + aₘ₂x₂ + ... + aₘₙxₙ = bₘ
```

Матричная форма: A·X = B

Формула словами: Матрица коэффициентов умножить на вектор неизвестных равно вектору свободных членов.

Пояснение к формуле:

· A = [aᵢⱼ] — матрица размера m×n (коэффициенты)
· X = [xⱼ] — вектор-столбец размера n×1 (неизвестные)
· B = [bᵢ] — вектор-столбец размера m×1 (свободные члены)

Пример:

```
2x + 3y = 8
4x - y = 2
```

В матричной форме:

```
[2  3] [x]   [8]
[4 -1] [y] = [2]
```

Виды СЛАУ:

1. Определённая: m = n, det(A) ≠ 0 → единственное решение
2. Неопределённая: rang(A) = rang(A|B) < n → бесконечно решений
3. Несовместная: rang(A) ≠ rang(A|B) → нет решений

Свойства:

1. Линейность: Если X₁ и X₂ — решения, то αX₁ + βX₂ — тоже решение
2. Суперпозиция: Решение линейно зависит от правой части
3. Единственность: Зависит от ранга матриц

Когда используется:

· В физике (законы Кирхгофа, статика)
· В экономике (модели Леонтьева)
· В компьютерной графике (преобразования)
· В машинном обучении (линейная регрессия)

Методы решения:

1. Точные: Гаусса, Крамера, обратной матрицы
2. Итерационные: Якоби, Зейделя, релаксации
3. Для специальных матриц: прогонка, квадратного корня

14. Теорема Кронекера-Капелли. Определение ранга матрицы

Определение ранга матрицы: Максимальное число линейно независимых строк (или столбцов).

Обозначение: rang(A) или rank(A)

Теорема Кронекера-Капелли: СЛАУ A·X = B совместна ⇔ rang(A) = rang(A|B)

Формулировка словами: Система имеет решение тогда и только тогда, когда ранг основной матрицы равен рангу расширенной матрицы.

Пояснение:

· A — основная матрица (коэффициенты)
· (A|B) — расширенная матрица (коэффициенты + свободные члены)
· Совместность — существование хотя бы одного решения

Способы нахождения ранга:

1. Метод элементарных преобразований: приведение к ступенчатому виду
2. Метод окаймляющих миноров: поиск ненулевого минора максимального порядка
3. Через линейную независимость строк/столбцов

Пример:

```
x + y = 3
2x + 2y = 6
```

· A = [[1,1],[2,2]], rang(A)=1
· (A|B) = [[1,1,3],[2,2,6]], rang(A|B)=1
· rang(A)=rang(A|B)=1 < 2 → бесконечно решений

Следствия теоремы:

1. rang(A) = rang(A|B) = n → единственное решение
2. rang(A) = rang(A|B) < n → бесконечно решений
3. rang(A) ≠ rang(A|B) → нет решений

Свойства ранга:

1. rang(A) ≤ min(m,n)
2. rang(Aᵀ) = rang(A)
3. rang(A·B) ≤ min(rang(A), rang(B))

Когда используется:

· При исследовании СЛАУ на совместность
· В теории линейных пространств
· В анализе данных (факторный анализ)

Практическое применение:

1. Проверить совместность системы перед решением
2. Определить количество свободных переменных
3. Найти базис системы векторов

15. Метод Гаусса

Определение: Метод последовательного исключения неизвестных из СЛАУ.

Суть метода: Приведение расширенной матрицы к ступенчатому виду элементарными преобразованиями строк.

Алгоритм:

```
1. ПРЯМОЙ ХОД:
   Для i = 1 до n-1:
     Выбрать ведущий элемент aᵢᵢ ≠ 0
     Для j = i+1 до n:
       Вычислить коэффициент m = aⱼᵢ / aᵢᵢ
       Вычесть из j-й строки i-ю, умноженную на m
       
2. ОБРАТНЫЙ ХОД:
   xₙ = bₙ / aₙₙ
   Для i = n-1 до 1:
     xᵢ = (bᵢ - Σ aᵢⱼxⱼ) / aᵢᵢ, где сумма от j=i+1 до n
```

Элементарные преобразования строк:

1. Перестановка двух строк
2. Умножение строки на число ≠ 0
3. Прибавление к одной строке другой, умноженной на число

Пример:

```
2x + 3y = 8   (1)
4x - y = 2    (2)
```

1. Вычитаем из (2) 2×(1): (4x-y) - 2(2x+3y) = 2 - 16 → -7y = -14 → y=2
2. Из (1): 2x + 6 = 8 → 2x = 2 → x=1

Матричная запись:

```
[2  3 | 8]   [2  3 | 8]
[4 -1 | 2] → [0 -7 |-14] → y=2, x=1
```

Свойства:

1. Точность: Даёт точное решение (без итераций)
2. Сложность: O(n³) операций
3. Устойчивость: Чувствителен к ошибкам округления
4. Определитель: det(A) = произведение диагональных элементов после прямого хода

Когда используется:

· Для решения СЛАУ небольшой размерности (n ≤ 1000)
· При вычислении обратной матрицы
· При нахождении определителя
· Вручную или на компьютере для точных вычислений

Модификации:

1. Метод Гаусса с выбором главного элемента — для улучшения устойчивости
2. Метод Гаусса-Жордана — приведение к диагональному виду
3. LU-разложение — факторизация матрицы

Преимущества:

· Универсальность (для любых совместных систем)
· Относительная простота
· Основа для многих других методов

Недостатки:

· Накопление ошибок округления
· Большое число операций для больших систем
· Требует хранения всей матрицы

16. Метод Якоби

Определение: Итерационный метод решения СЛАУ, в котором каждое уравнение решается относительно одной переменной.

Формула: xᵢ⁽ᵏ⁺¹⁾ = (bᵢ - Σ aᵢⱼ·xⱼ⁽ᵏ⁾) / aᵢᵢ, где сумма по j ≠ i

Формула словами: Новое значение i-й переменной равно (свободный член минус сумма произведений коэффициентов на старые значения других переменных), делённое на диагональный коэффициент.

Матричная форма: X⁽ᵏ⁺¹⁾ = D⁻¹·(B - R·X⁽ᵏ⁾)
где A= D + R, D — диагональная часть A, R — остаток

Суть метода: Последовательное уточнение решения, используя значения переменных с предыдущей итерации.

Пояснение к формуле:

· xⱼ⁽ᵏ⁾ — значения переменных на k-й итерации
· aᵢⱼ — коэффициенты матрицы
· bᵢ — свободные члены
· Деление на aᵢᵢ — требует диагонального преобладания

Пример (система 2×2):

```
10x + y = 11  → x = (11 - y)/10
x + 10y = 11  → y = (11 - x)/10
```

Итерации с x⁽⁰⁾=0, y⁽⁰⁾=0:

```
k=1: x=1.1, y=1.1
k=2: x=(11-1.1)/10=0.99, y=(11-1.1)/10=0.99
k=3: x=(11-0.99)/10=1.001, y=(11-0.99)/10=1.001
...
```

Алгоритм:

```
1. Задать начальное приближение X⁽⁰⁾
2. Для k = 0,1,2,...:
   Для i = 1 до n:
     xᵢ⁽ᵏ⁺¹⁾ = (bᵢ - Σ aᵢⱼ·xⱼ⁽ᵏ⁾) / aᵢᵢ
   Проверить критерий остановки
```

Свойства:

1. Простота реализации: Не требует хранения предыдущих значений кроме X⁽ᵏ⁾
2. Параллелизуемость: Каждая переменная вычисляется независимо
3. Скорость сходимости: Медленная (линейная)
4. Память: O(n) дополнительной памяти

Когда используется:

· Для больших разреженных систем
· Когда матрица имеет диагональное преобладание
· В параллельных вычислениях
· Для начального приближения в других методах

Условия сходимости:

1. Строгое диагональное преобладание: |aᵢᵢ| > Σ|aᵢⱼ|, j≠i
2. Матрица симметричная и положительно определённая
3. Спектральный радиус итерационной матрицы < 1

Критерий остановки:

1. По аргументу: max|xᵢ⁽ᵏ⁺¹⁾ - xᵢ⁽ᵏ⁾| < ε
2. По невязке: ||A·X⁽ᵏ⁾ - B|| < ε
3. По числу итераций: k > max_iter

Модификации:

1. Метод Зейделя: Использует уже вычисленные новые значения
2. Метод релаксации: Введение параметра релаксации ω
3. Блочный метод Якоби: Для блочных матриц

Преимущества:

· Простота программирования
· Малое потребление памяти
· Устойчивость для диагонально доминирующих систем

Недостатки:

· Медленная сходимость
· Требует диагонального преобладания
· Может не сходиться для произвольных систем

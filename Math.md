1. ТРАНСЦЕНДЕНТНЫЕ УРАВНЕНИЯ - ПОДРОБНО

Что значит "трансцендентный"?

Слово происходит от латинского "transcendere" - "переступать", "выходить за пределы".

Трансцендентное уравнение - такое уравнение, которое НЕЛЬЗЯ свести к алгебраическому путём алгебраических преобразований.

Ключевое отличие от алгебраических уравнений:

Алгебраические Трансцендентные
Содержат только +, -, ×, ÷, степени с целым показателем Содержат "неалгебраические" функции
Пример: 3x² - 2x + 5 = 0 Пример: sin(x) + eˣ = 2
Всегда можно записать как P(x) = 0 Нельзя записать как P(x) = 0
Число корней ≤ степени многочлена Может быть бесконечно много корней

Типы трансцендентных функций в уравнениях:

1. Тригонометрические:
   · sin(x), cos(x), tan(x), и их обратные
   · Пример реальной задачи: Определить угол отклонения маятника: sin(θ) = (g/L)·t²
2. Показательные:
   · eˣ, aˣ (где a > 0, a ≠ 1)
   · Пример реальной задачи: Радиоактивный распад: N(t) = N₀·e^(-λt)
3. Логарифмические:
   · ln(x), logₐ(x)
   · Пример реальной задачи: pH раствора: pH = -log₁₀[H⁺]
4. Гиперболические:
   · sinh(x), cosh(x), tanh(x)
   · Пример реальной задачи: Форма висящего кабеля (цепная линия): y = a·cosh(x/a)
5. Комбинации:
   · eˣ + sin(x) = 0
   · ln(x) - x² = 1

Особенности решения трансцендентных уравнений:

· Нет общего метода решения (в отличие от квадратных уравнений)
· Часто имеют бесконечное число корней (особенно с периодическими функциями)
· Могут не иметь аналитического решения - только численные методы
· Требуют осторожности с областью определения (логарифм - только положительные числа)

Пример из физики:

Уравнение колебаний с трением:
m·x''(t)+ c·x'(t) + k·x(t) = 0
Характеристическое уравнение:m·r² + c·r + k = 0 (алгебраическое)

Но если трение зависит от скорости сложным образом:
m·x''+ arctan(x') + k·sin(x) = 0 - уже трансцендентное!

2. ЧТО ТАКОЕ ε (ЭПСИЛОН)?

Математическое и физическое значение:

ε (эпсилон) - греческая буква, в математике обозначает:

1. Малое положительное число
2. Точность вычислений
3. Допустимую ошибку

Зачем нужна точность ε?

Представьте, что вы измеряете длину комнаты:

· Рулеткой с сантиметровыми делениями: ε ≈ 0.5 см
· Лазерным дальномером: ε ≈ 0.1 мм
· Нанотехнологиями: ε ≈ 0.000001 мм

В вычислениях аналогично: мы решаем, какая точность нас устраивает.

Как выбирается ε?

Зависит от задачи:

1. Инженерные расчеты: ε = 0.001 (часто достаточно)
2. Научные исследования: ε = 10⁻⁶ или меньше
3. Финансовые расчеты: ε = 0.01 (копейка/цент)
4. Компьютерная графика: ε = 0.0001 (пиксельная точность)

ε в численных методах - ПРАКТИЧЕСКИ:

Пример для метода дихотомии:
Ищем корень уравнения с точностью 0.001

```python
# Псевдокод:
epsilon = 0.001  # Наша требуемая точность

while abs(b - a) > epsilon:  # Пока длина отрезка больше точности
    c = (a + b) / 2
    if f(c) == 0:
        break
    elif f(a) * f(c) < 0:
        b = c
    else:
        a = c
```

Два основных использования ε:

1. По аргументу: |x_new - x_old| < ε
   · "Новое приближение отличается от старого менее чем на ε"
   · Пример: Ищем корень, получили 1.2345 и 1.2347, ε=0.001 ⇒ продолжаем
2. По функции: |f(x)| < ε
   · "Значение функции близко к нулю с точностью ε"
   · Пример: f(1.234) = -0.0008, ε=0.001 ⇒ считаем корнем

Важное замечание:

Иногда используют относительную точность:
|(x_new- x_old) / x_new| < ε
Это полезно,когда значения очень большие или очень маленькие.

3. ПРОИЗВОДНАЯ - ДЕТАЛЬНОЕ ОБЪЯСНЕНИЕ

Интуитивное понимание:

Производная - это:

1. Скорость изменения функции
2. Угловой коэффициент касательной
3. Мгновенная скорость (в физике)

Геометрическая интерпретация:

Пример: Автомобиль едет по дороге.

· Положение в момент t: x(t)
· Средняя скорость за час: (x(конец) - x(начало)) / 1 час
· Мгновенная скорость в момент t: производная x'(t)

концептуально

```
f(x)
   |       /
   |      /
   |     /.
   |    / .
   |   /  .  ← Касательная: наклон = f'(x₀)
   |  /   .
   | /    .
   |/_____._.___→ x
          x₀
```

Формальное определение:

Производная функции f в точке x₀:

```
          f(x₀ + h) - f(x₀)
f'(x₀) = lim ──────────────
         h→0       h
```

Словами: Предел отношения приращения функции к приращению аргумента, когда приращение аргумента стремится к нулю.

Пример вычисления:

f(x) = x²

```
f'(x) = lim [(x+h)² - x²] / h
        h→0

       = lim [x² + 2xh + h² - x²] / h
         h→0
       
       = lim (2xh + h²) / h
         h→0
       
       = lim (2x + h) = 2x
         h→0
```

Производные основных функций:

1. Константа: (c)' = 0
2. Степенная: (xⁿ)' = n·xⁿ⁻¹
3. Экспонента: (eˣ)' = eˣ
4. Логарифм: (ln x)' = 1/x
5. Синус: (sin x)' = cos x
6. Косинус: (cos x)' = -sin x

Правила дифференцирования:

1. Сумма: (f + g)' = f' + g'
2. Произведение: (f·g)' = f'·g + f·g'
3. Частное: (f/g)' = (f'·g - f·g')/g²
4. Цепное правило: (f(g(x)))' = f'(g(x))·g'(x)

Производная в методе Ньютона - КОНКРЕТНО:

Метод Ньютона: x_new = x_old - f(x_old)/f'(x_old)

Почему именно так?

1. Имеем точку (x₀, f(x₀))
2. Проводим касательную: y = f(x₀) + f'(x₀)·(x - x₀)
3. Ищем пересечение с осью OX: 0 = f(x₀) + f'(x₀)·(x - x₀)
4. Решаем: x = x₀ - f(x₀)/f'(x₀)

4. МАТРИЧНАЯ ФОРМА СЛАУ - ПОДРОБНО

Что такое матрица в контексте СЛАУ?

Матрица - это не просто таблица, а структурированный способ хранения коэффициентов системы.

Переход от системы к матрице:

Дана система:

```
2x + 3y - z = 5    ... (1)
4x - y + 2z = 3    ... (2)
x + 2y + z = 4     ... (3)
```

Шаг 1: Выписываем коэффициенты при неизвестных:
Из(1): 2 при x, 3 при y, -1 при z, свободный член 5
Из(2): 4 при x, -1 при y, 2 при z, свободный член 3
Из (3): 1 при x, 2 при y, 1 при z, свободный член 4

Шаг 2: Формируем матрицу коэффициентов A:

```
    [ 2   3  -1 ]
A = [ 4  -1   2 ]
    [ 1   2   1 ]
```

Шаг 3: Формируем вектор неизвестных X:

```
    [ x ]
X = [ y ]
    [ z ]
```

Шаг 4: Формируем вектор свободных членов B:

```
    [ 5 ]
B = [ 3 ]
    [ 4 ]
```

Матричная форма: A·X = B

Что значит A·X? Это умножение матрицы на вектор.

5. УМНОЖЕНИЕ МАТРИЦЫ НА ВЕКТОР - ДЕТАЛЬНО

Правило умножения:

Чтобы умножить матрицу A (размер m×n) на вектор X (размер n×1):

1. Берем i-ю строку матрицы A
2. Умножаем поэлементно на элементы вектора X
3. Складываем результаты

Формула: (A·X)[i] = Σ A[i,j]·X[j], где сумма по j=1..n

Пример для нашей системы:

```
A = [2  3 -1]   X = [x]   B = [5]
    [4 -1  2]       [y]       [3]
    [1  2  1]       [z]       [4]
```

Вычисляем A·X:

1. Первая строка: 2·x + 3·y + (-1)·z = 2x + 3y - z
2. Вторая строка: 4·x + (-1)·y + 2·z = 4x - y + 2z
3. Третья строка: 1·x + 2·y + 1·z = x + 2y + z

Получаем систему уравнений! Именно ту, с которой начинали.

Умножение матриц общего вида:

Если A (m×n) и B (n×p), то C = A·B будет (m×p):

```
C[i,k] = Σ A[i,j]·B[j,k]
         j=1..n
```

Пример:

```
A = [1 2]   B = [5 6]
    [3 4]       [7 8]
    
C = A·B:
C[1,1] = 1·5 + 2·7 = 5 + 14 = 19
C[1,2] = 1·6 + 2·8 = 6 + 16 = 22
C[2,1] = 3·5 + 4·7 = 15 + 28 = 43
C[2,2] = 3·6 + 4·8 = 18 + 32 = 50
```

Важные свойства умножения матриц:

1. НЕ коммутативно: A·B ≠ B·A (в общем случае)
2. Ассоциативно: (A·B)·C = A·(B·C)
3. Дистрибутивно: A·(B+C) = A·B + A·C

Единичная матрица (аналог 1 для матриц):

```
I = [1 0 0]
    [0 1 0]
    [0 0 1]
```

Для любой матрицы A: A·I = I·A = A

6. МЕТОД ЯКОБИ - ПОДРОБНЫЙ РАЗБОР

Физическая интерпретация метода:

Представьте 3 комнаты с разной температурой. В каждой комнате есть термостат, который:

1. Измеряет температуру в соседних комнатах
2. Устанавливает свою температуру как среднюю от соседних

Итерация - один цикл обновления всех термостатов.

Математическая постановка:

Дана система:

```
a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ = b₁
a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ = b₂
...
aₙ₁x₁ + aₙ₂x₂ + ... + aₙₙxₙ = bₙ
```

Преобразование к виду Якоби:

Из i-го уравнения выражаем xᵢ:

```
xᵢ = (bᵢ - Σ aᵢⱼxⱼ) / aᵢᵢ
         j≠i
```

Матричная запись:

Пусть A = D + R, где:

· D - диагональная часть A (только элементы aᵢᵢ)
· R - остаток (все элементы кроме диагональных)

Тогда:
A·X= (D + R)·X = D·X + R·X = B

Отсюда:
D·X= B - R·X
X= D⁻¹·(B - R·X)  ← Формула Якоби

Поэлементная запись:

```
xᵢ⁽ᵏ⁺¹⁾ = (bᵢ - Σ aᵢⱼ·xⱼ⁽ᵏ⁾) / aᵢᵢ
              j≠i
```

Ключевой момент: Для вычисления xᵢ⁽ᵏ⁺¹⁾ используются ТОЛЬКО значения с предыдущей итерации xⱼ⁽ᵏ⁾.

ПОЛНЫЙ пример решения системы методом Якоби:

Система:

```
10x + y + z = 12   ... (1)
x + 10y + z = 12   ... (2)
x + y + 10z = 12   ... (3)
```

Шаг 1: Приводим к виду Якоби:
Из(1): x = (12 - y - z) / 10
Из(2): y = (12 - x - z) / 10
Из (3): z = (12 - x - y) / 10

Шаг 2: Начальное приближение:
x⁽⁰⁾= 0, y⁽⁰⁾ = 0, z⁽⁰⁾ = 0

Шаг 3: Итерации:

Итерация 1:
x⁽¹⁾= (12 - 0 - 0)/10 = 1.2
y⁽¹⁾= (12 - 0 - 0)/10 = 1.2
z⁽¹⁾= (12 - 0 - 0)/10 = 1.2

Итерация 2:
x⁽²⁾= (12 - 1.2 - 1.2)/10 = (9.6)/10 = 0.96
y⁽²⁾= (12 - 1.2 - 1.2)/10 = 0.96
z⁽²⁾= (12 - 1.2 - 1.2)/10 = 0.96

Итерация 3:
x⁽³⁾= (12 - 0.96 - 0.96)/10 = (10.08)/10 = 1.008
y⁽³⁾= 1.008
z⁽³⁾= 1.008

Итерация 4:
x⁽⁴⁾= (12 - 1.008 - 1.008)/10 = (9.984)/10 = 0.9984
...

Сходится к решению: x = 1, y = 1, z = 1

Почему сходится?

Матрица системы имеет диагональное преобладание:

· |10| > |1| + |1| (10 > 2) ✓
· |10| > |1| + |1| (10 > 2) ✓
· |10| > |1| + |1| (10 > 2) ✓

Условие сходимости Якоби:

1. Строгое диагональное преобладание:
   |aᵢᵢ| > Σ |aᵢⱼ| для всех i
   j≠i
2. ИЛИ: Матрица симметричная и положительно определённая

Критерий остановки:

1. По аргументу: max|xᵢ⁽ᵏ⁺¹⁾ - xᵢ⁽ᵏ⁾| < ε
2. По невязке: ||A·X⁽ᵏ⁾ - B|| < ε
3. По времени/итерациям: k > max_iterations

7. ФОРМУЛЫ ПРЯМОУГОЛЬНИКОВ И ТРАПЕЦИЙ - ПОДРОБНЫЙ РАЗБОР

ГЕОМЕТРИЧЕСКИЙ СМЫСЛ ИНТЕГРАЛА:

Задача: Найти площадь под кривой y = f(x) на отрезке [a, b]

Проблема: Кривая не прямая, формула "длина × ширина" не работает.

Решение: Разбить на маленькие участки, где кривая почти прямая.

---

ФОРМУЛА ПРЯМОУГОЛЬНИКОВ:

Общая идея:

Заменить криволинейную трапецию несколькими прямоугольниками.

Алгоритм:

1. Разбиваем отрезок [a, b] на n равных частей
   Шаг: h = (b - a) / n
   Точки: xᵢ = a + i·h, где i = 0, 1, ..., n
2. На каждом отрезке [xᵢ, xᵢ₊₁] выбираем высоту прямоугольника

Три варианта:

1. Левые прямоугольники:

```
Площадь i-го прямоугольника: h·f(xᵢ)
Общая площадь: S ≈ h·[f(x₀) + f(x₁) + ... + f(xₙ₋₁)]
```

```
    f(x)
     |     _____
     |    |  _ |
     |    | | | |
     |    | | | |
     |____|_|_|_|______ x
     x₀  x₁ x₂ x₃
```

2. Правые прямоугольники:

```
Площадь i-го прямоугольника: h·f(xᵢ₊₁)
Общая площадь: S ≈ h·[f(x₁) + f(x₂) + ... + f(xₙ)]
```

3. Средние прямоугольники:

```
Площадь i-го прямоугольника: h·f((xᵢ + xᵢ₊₁)/2)
Общая площадь: S ≈ h·[f((x₀+x₁)/2) + ... + f((xₙ₋₁+xₙ)/2)]
```

ПОЛНЫЙ пример вычисления:

Вычислим ∫[0,1] x² dx формулой левых прямоугольников (n=4)

1. Параметры:
   · a = 0, b = 1, n = 4
   · h = (1-0)/4 = 0.25
   · Точки: 0, 0.25, 0.5, 0.75, 1
2. Значения функции:
   · f(0) = 0² = 0
   · f(0.25) = 0.25² = 0.0625
   · f(0.5) = 0.5² = 0.25
   · f(0.75) = 0.75² = 0.5625
   · f(1) = 1² = 1 (не нужно для левых прямоугольников)
3. Площади прямоугольников:
   · [0, 0.25]: 0.25 × 0 = 0
   · [0.25, 0.5]: 0.25 × 0.0625 = 0.015625
   · [0.5, 0.75]: 0.25 × 0.25 = 0.0625
   · [0.75, 1]: 0.25 × 0.5625 = 0.140625
4. Сумма: 0 + 0.015625 + 0.0625 + 0.140625 = 0.21875

Точное значение: ∫[0,1] x² dx = [x³/3]₀¹ = 1/3 ≈ 0.3333
Погрешность:0.3333 - 0.21875 = 0.11455

ФОРМУЛА ТРАПЕЦИЙ:

Общая идея:

Заменить криволинейную трапецию несколькими трапециями (лучше, чем прямоугольники, т.к. учитывает наклон).

Алгоритм:

1. Разбиваем отрезок [a, b] на n равных частей
   h = (b - a) / n
   xᵢ = a + i·h
2. На каждом отрезке [xᵢ, xᵢ₊₁] строим трапецию

Вывод формулы:

Площадь одной трапеции:
Основания:f(xᵢ) и f(xᵢ₊₁), высота: h

```
Площадь трапеции = (основание₁ + основание₂) × высота / 2
Sᵢ = (f(xᵢ) + f(xᵢ₊₁)) × h / 2
```

Суммируем все трапеции:

```
S = Σ Sᵢ = h/2 × Σ [f(xᵢ) + f(xᵢ₊₁)]
            i=0..n-1

Раскрываем сумму:
S = h/2 × [(f(x₀)+f(x₁)) + (f(x₁)+f(x₂)) + ... + (f(xₙ₋₁)+f(xₙ))]

Замечаем, что f(x₁), f(x₂), ..., f(xₙ₋₁) встречаются ДВАЖДЫ:
S = h/2 × [f(x₀) + 2f(x₁) + 2f(x₂) + ... + 2f(xₙ₋₁) + f(xₙ)]
```

Итоговая формула:

```
     [ f(x₀) + f(xₙ)     ]
S = h| ————————————— + f(x₁) + f(x₂) + ... + f(xₙ₋₁) |
     [       2           ]
```

ГЕОМЕТРИЧЕСКАЯ ИНТЕРПРЕТАЦИЯ:

```
    f(x)
     |       /|
     |      / |
     |     /  | ← Трапеция
     |    /   |
     |   /    |
     |__/_____|____ x
     xᵢ      xᵢ₊₁
```

Каждая трапеция лучше аппроксимирует кривую, чем прямоугольник, потому что учитывает изменение функции на отрезке.

ПОЛНЫЙ пример вычисления:

Вычислим ∫[0,1] x² dx формулой трапеций (n=4)

1. Параметры:
   · a = 0, b = 1, n = 4
   · h = 0.25
   · Точки: 0, 0.25, 0.5, 0.75, 1
2. Значения функции:
   · f(0) = 0² = 0
   · f(0.25) = 0.0625
   · f(0.5) = 0.25
   · f(0.75) = 0.5625
   · f(1) = 1
3. Применяем формулу:
   ```
   S = 0.25 × [ (0 + 1)/2 + 0.0625 + 0.25 + 0.5625 ]
     = 0.25 × [ 0.5 + 0.0625 + 0.25 + 0.5625 ]
     = 0.25 × [ 0.5 + 0.875 ]
     = 0.25 × 1.375
     = 0.34375
   ```
4. Сравнение:
   · Точное значение: 0.3333
   · Метод трапеций: 0.34375 (погрешность 0.01045)
   · Левые прямоугольники: 0.21875 (погрешность 0.11455)

Вывод: Метод трапеций точнее в 11 раз!

КОГДА КАКОЙ МЕТОД ИСПОЛЬЗОВАТЬ?

1. Прямоугольники:
   · Быстрая оценка "на коленке"
   · Когда точность не критична
   · Для монотонных функций (всегда дают оценку сверху или снизу)
2. Трапеции:
   · Инженерные расчеты (обычно достаточно)
   · Когда функция гладкая
   · Баланс точности и простоты
3. Более точные методы (не в билете, но для контекста):
   · Симпсона (параболы): для очень точных расчетов
   · Монте-Карло: для многомерных интегралов

ПРАКТИЧЕСКИЙ СОВЕТ:

Для оценки погрешности часто считают интеграл дважды:

· С n отрезками
· С 2n отрезками
  Если разница мала- результат надежен.

8. ДОПОЛНИТЕЛЬНЫЙ КОНТЕКСТ И СВЯЗИ

Связь между методами:

1. Локализация → Численные методы:
   Сначала находим, где примерно корень, потом уточняем
2. Метод Ньютона → Производные:
   Нужно уметь вычислять производную функции
3. Метод Якоби → Диагональное преобладание:
   Если нет - метод может не сойтись
4. Интегрирование → Аппроксимация:
   Все методы численного интегрирования основаны на аппроксимации функции более простыми фигурами

В программировании:
